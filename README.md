
## Customer Care:
Is an application that helps any user get access to a firms's available customer care's lines and respective 
branches based on the user's geolocation.
User clicks on the helpline button on the home-page, user's permission is requested to fetch user's location, 
if granted, it returns availble customer care line if any exists that matches user's city and country 
and if none exists, it returns a contact-us form.
